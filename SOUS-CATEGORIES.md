## 🔧 UTILISATION DANS LE CODE

### Récupérer toutes les sous-catégories d'un thème

```javascript
import { getSubcategories } from './data/ce1dThemes.js';

const numbersSubcats = getSubcategories('numbers');
// Retourne: [
//   { code: 'N1', name: { fr: 'Calcul numérique', ... } },
//   { code: 'N2', name: { fr: 'Calcul littéral', ... } },
//   ...
// ]
```

### Récupérer une sous-catégorie par son code

```javascript
import { getSubcategoryByCode } from './data/ce1dThemes.js';

const subcat = getSubcategoryByCode('N1');
// Retourne: {
//   code: 'N1',
//   name: { fr: 'Calcul numérique', ... },
//   themeId: 'numbers',
//   themeColor: '#3B82F6'
// }
```

### Récupérer toutes les sous-catégories

```javascript
import { getAllSubcategories } from './data/ce1dThemes.js';

const allSubcats = getAllSubcategories();
// Retourne un tableau de 18 sous-catégories avec leurs informations complètes
```
